# Missão Prenex

Jogo didático em HTML, CSS e JavaScript puro sobre Forma Normal Prenex em lógica de predicados.

## Como usar

Abra `index.html` no navegador ou publique a pasta no GitHub Pages. O jogo pede o nome do aluno e usa fases de montagem: o aluno arrasta ou clica em peças de quantificadores e matriz para preencher os slots da forma prenex.

O placar considera fases concluídas, vidas restantes, combo, uso de dicas e erros.

## Publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie estes arquivos para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione `main` e a pasta `/root`.
6. Salve e aguarde o link do Pages.

## Placar

O GitHub Pages é estático e não possui banco de dados próprio. Por isso, o ranking fica salvo no navegador de quem jogou. Para consolidar resultados:

- cada aluno pode clicar em `Copiar resultado` e enviar a você;
- em um computador compartilhado, o ranking fica acumulado localmente;
- você pode usar `Exportar CSV` e `Importar CSV` para juntar resultados de vários navegadores.

## Conteúdo da aula

O jogo cobre:

- definição de Forma Normal Prenex;
- prefixo de quantificadores e matriz;
- conjunção e disjunção;
- negação com inversão de quantificadores;
- implicação;
- renomeação de variáveis para evitar erro;
- desafio final de montagem com variável repetida.
